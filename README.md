## 🚀 Tech Stack

| Technology  | Version  |
| ----------- | -------- |
| Next.js     | 15.2.1   |
| React       | 19.0.0   |
| TypeScript  | ^5       |
| Prisma ORM  | ^6.5.0   |
| NextAuth.js | ^4.24.11 |
| Sass        | ^1.85.1  |

<br>

## 📦 Libraries

- **Prisma** – Database ORM 🗄️
- **NextAuth.js** – Authentication 🔐
- **bcrypt** – Password hashing 🔑
- **React Hook Form** – Form handling 📝
- **Sass** – Stylish styling 💅
- **ESLint** – Code linting 🚨

## 📜 Scripts

Run these commands to get started:

```sh
# Start dev server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Lint code
npm run lint
```
